
Замечания по конструкции проекта.
Прокт состоит из 2 частей

Объекты мз ltcore осуществляют всю работу по обработке данных 

Объекты из ltgui создают графический интерфейс

Объекты ltcore ничего не знают о существовании графических и только посылают сигналы
Графические объекты при  инициализации получают ссылку на объекты ltcore
используют ее для установки соединений сигнал-слот
также могут напрямую вызывать их методы 
Желательно проектировать их так, чтобы каждый графический объект 
взаимодействовал только с одним элементом из ltCore

Текущие классы, приведенные к этой схеме

cvPlayer -- плейер. ответственен за чтение данных 
интерфейс -- videoWidget
