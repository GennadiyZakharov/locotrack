Locotrack 1.0

Программа предназначена для анализа локомоции свободно движущегося интактного животного

Порядок работы и возможности программы

1. В качестве источника информации используется видеофайл в любом формате, декодируемом OpenCV, либо вход с камеры (если хватает производительности)

2. Открытый видеофайл можно просматривать по отдельным кадрам, перематывать, проигрывать c нормальной скоростью.

3. Детектируется светлый объект на темном фоне. Если требуется определять темный объект на светлом фоне,
изображение инвертируется.

3. Перед началом регистрации экспериментатор вручную определяет границы камер.
Количество камер не ограничено. 
Каждая камера должна содержать только одно животное.


4. Экпериментатор задает масштаб (по умолчанию 20 мм).

5. Как только камера задана, начинается отслеживание объекта.

Изображение проходит следующие стади обработки:
- Инвертирование (для отслеживания темных объектов)
- Преобразование к ч/б (можно сделать отдельно по каналам)

----- Все параметры объекта определяются и записываются в экранных координатах ------

----- Все остальное делается для каждой ячейки в отдельности ------  
- Определение точек с минимальной и максимальной яркостью, и их положения
- Выбор значения порога (по умолчанию 60% * (максимальная яркость-минимальная яркость))
- Обрезка по порогу (все точки, яркость которых меньше порога, сбрасываются в 0)
- Определение контура изображения 
- Подсчит ценра масс объекта 
- Запись всех определенных параметров в объект ltObject.

6. На изображении отображаются камеры, контур объекта, точки максимальной яркости и центр масс.
Координатой обекта считаются координаты центра масс, если они определены.
Иначе -- координаты самой яркой точки.     

7. При необходимости можно включить показ обработанного изображения (чтобы оценить работу пороговой функции)

8. По умолчанию определенные координаты никуда не записыаются. На панели анализа траектории 
находится кнопка "запись траектории"
При ее нажатии начинается запись траектории от текужего положения и до конца файла. 


===== Работа в программе ====
Основной рабочей единицей является проект. Программа позволяет одновременно работать только с одним проектом.
На диске проект представляет собой каталог, в котором хранятся все ринадлежащие проекту файлы. Можно также сделать архивацию этого каалога.
